# Changelog

All notable changes to this project are documented here. The project follows
[Semantic Versioning](https://semver.org/).

## [0.1.1] - 2026-07-18

### Fixed

- native Hugging Face Spaces installation now receives the exact pinned runtime
  dependency set directly through `requirements.txt`

## [0.1.0] - 2026-07-18

### Added

- reproducible runtime and development dependency locks
- one local quality command with formatting, linting, boundary type checks,
  compile validation, deterministic tests, and preflight checks
- security policy, AI threat model, contribution guidance, and pull-request
  template
- least-privilege, immutable CI workflow and dependency-update automation

### Documented

- supported Python versions, maturity, architecture, privacy boundaries,
  limitations, and release process
